# opnsense_quagga_ospf (Resource)

~> **Singleton Resource** This resource manages existing upstream Quagga configuration that cannot be created or destroyed through Terraform. You **must** import it before managing. Running `terraform destroy` removes the resource from state only and does **not** modify the upstream configuration.

Manages Quagga OSPF (Open Shortest Path First) global settings. This is a singleton resource that manages existing upstream configuration.

**Important:** This resource must be imported before it can be managed:
```bash
terraform import opnsense_quagga_ospf.ospf quagga_ospf
```

After importing, you can manage the configuration with `terraform apply`. Running `terraform destroy` will remove the resource from state but will NOT modify the upstream configuration.

## Example Usage

```terraform

resource "opnsense_quagga_ospf" "ospf" {
  enabled   = true
  router_id = "10.0.0.1"
```

## Schema

### Optional

- `carp_demote` (Boolean) Increase the CARP demotion counter while OSPF is not fully converged, causing CARP to prefer the peer until OSPF is ready. Defaults to `false`.
- `cost_reference` (Number) Reference bandwidth in Mbps used to calculate interface cost. Interfaces with a speed of `cost_reference` Mbps receive a cost of 1. Use `-1` to leave unset.
- `enabled` (Boolean) Enable the OSPF routing daemon. Defaults to `false`.
- `log_adjacency_changes` (Boolean) Log a message when an OSPF adjacency changes state. Defaults to `false`.
- `originate` (Boolean) Originate and advertise an OSPF default route (type-5 LSA) into the OSPF domain. Defaults to `false`.
- `originate_always` (Boolean) Always originate the default route even when no default route exists in the routing table. Requires `originate` to be `true`. Defaults to `false`.
- `originate_metric` (Number) Metric value assigned to the originated default route. Use `-1` to leave unset.
- `passive_interfaces` (Set of String) Set of interface names to make passive. Passive interfaces do not send or receive OSPF Hello packets but still advertise their connected networks.
- `router_id` (String) OSPF router ID in IPv4 dotted-decimal notation (e.g. `"10.0.0.1"`). Leave empty to use the highest interface IP address.

### Read-Only

- `id` (String) Always set to `quagga_ospf`. Use this value when importing: `terraform import opnsense_quagga_ospf.ospf quagga_ospf`

## Import
