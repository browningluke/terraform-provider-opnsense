# opnsense_trust_ca (Resource)

Manages a Certificate Authority in the OPNsense Trust store.

## Example Usage

```terraform
# Create an internal root CA
resource "opnsense_trust_ca" "root" {
  description  = "Internal Root CA"
  action       = "internal"
  key_type     = "4096"
  digest       = "sha256"
  lifetime     = "3650"
  country      = "US"
  organization = "Acme Corp"
  common_name  = "Acme Internal Root CA"
}

# Create an intermediate CA signed by the root CA
resource "opnsense_trust_ca" "intermediate" {
  description  = "Internal Intermediate CA"
  action       = "internal"
  key_type     = "2048"
  digest       = "sha256"
  lifetime     = "1825"
  caref        = opnsense_trust_ca.root.ref_id
  country      = "US"
  organization = "Acme Corp"
  common_name  = "Acme Intermediate CA"
}
```

## Schema

### Required

- `common_name` (String) Common name (CN) for the CA certificate.

### Optional

- `action` (String) Creation action: `internal` (generate a new CA), `existing` (import an existing CA), or `ocsp` (create an OCSP signing certificate). Defaults to `internal`.
- `caref` (String) Reference ID (`ref_id`) of the parent CA. Used for intermediate CAs or OCSP signers. Leave empty for root CAs.
- `city` (String) City or locality name.
- `country` (String) ISO 3166-1 alpha-2 country code (e.g. `US`, `CA`, `DE`).
- `crt` (String) Base64-encoded PEM certificate body. Required when `action` is `existing`. Computed (generated by OPNsense) when `action` is `internal`.
- `description` (String) Optional description for this CA.
- `digest` (String) Digest algorithm: `sha224`, `sha256` (default), `sha384`, `sha512`. Defaults to `sha256`.
- `email` (String) Email address for the CA subject.
- `key_type` (String) Key type and size: `512`, `1024`, `2048` (default), `3072`, `4096`, `8192`, `prime256v1`, `secp384r1`, `secp521r1`. Defaults to `2048`.
- `lifetime` (String) CA validity period in days. Defaults to `3650` (10 years). Note: OPNsense does not return the configured lifetime on subsequent reads; the value stored in Terraform state is preserved.
- `ocsp_uri` (String) OCSP responder URI embedded in issued certificates.
- `organization` (String) Organisation name.
- `organizational_unit` (String) Organisational unit name.
- `prv` (String, Sensitive) Base64-encoded PEM private key. Computed when `action` is `internal`; optionally provided when `action` is `existing`.
- `serial` (String) Next serial number for certificates issued by this CA.
- `state` (String) State or province name.

### Read-Only

- `crt_payload` (String) Decoded PEM certificate body (read-only).
- `id` (String) UUID of the Certificate Authority.
- `name` (String) Distinguished name string of the CA (read-only).
- `prv_payload` (String, Sensitive) Decoded PEM private key (read-only, sensitive).
- `ref_id` (String) Short hex reference ID used by other OPNsense subsystems (e.g. OpenVPN, IPsec) to reference this CA.
- `valid_from` (String) Unix timestamp of certificate start date (read-only).
- `valid_to` (String) Unix timestamp of certificate expiry date (read-only).
